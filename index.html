Footer Ads sample implementations

Highfivve's ad stack offers two possibilities for sticky footer ads:

  • a mobile sticky ad,
  • a sticky desktop footer ad.
  • a sticky footer ad v2 (desktop & mobile).
  • a sticky header ad with a sticky navbar.
  • and a sticky header ad with a navbar that's not sticky.

As there's some code that needs to be included in the publisher's website, this repository offers a sample implementation for each of these formats.
